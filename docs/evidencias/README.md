# Evidencias de Pasantía (pendiente)

Esta carpeta contiene los archivos de evidencia formal requeridos por la Universidad de Córdoba para el proceso de pasantía.

## Tipos de evidencia a incluir

| Tipo                 | Descripción                                                           |
| -------------------- | --------------------------------------------------------------------- |
| Capturas de pantalla | Evidencia visual del avance en el desarrollo del sistema              |
| Documentos `.docx`   | Versiones formales de la documentación técnica para entrega académica |
| Reportes de avance   | Informes semanales de progreso según el cronograma del plan           |
| Informe final        | Documento final de la pasantía                                        |

## Organización (prototipo)

```
evidencias/
├── semana-01/
│   ├── acta-reunion-YYYY-MM-DD.docx
│   └── capturas/
├── semana-02/
├── ...
└── informe-final/
    └── SIPAc-InformeFinal-CarlosCanabal.docx
```
